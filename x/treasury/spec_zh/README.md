## 摘要

财政部模块充当 Terra 经济的“中央银行”，通过[观察指标](./01_concepts.md#Observed-Indicators) 和调整[货币政策杠杆](./01_concepts.md#Monetary-) 来衡量宏观经济活动政策杠杆）以调节矿工激励以实现稳定的长期增长。

> 在财政部通过调整奖励来稳定矿工需求的同时，[市场](../market/spec/README.md) 负责通过套利和做市商来稳定 Terra 的价格。
